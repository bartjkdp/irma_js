Issuing test from QR

This demonstrates an alternative way to start an IRMA session: 

IRMA.issueFromQr(sessionPackage, success_cb, cancel_cb, failure_cb);
where sessionPackage is the contents of an IRMA issuance QR as returned by the API server's issuance creation call (instead of IRMA.issue(jwt, success_cb, cancel_cb, failure_cb), which also creates the session at the API server.)

For disclosing and signing, there are equivalent functions IRMA.verifyFromQr(qr, success_cb, cancel_cb, failure_cb) and IRMA.signFromQr(qr, success_cb, cancel_cb, failure_cb), respectively.
